# ComplyAI — Tech Stack

## Frontend
React + Tailwind CSS
Next.js (routing + API layer)

## Backend
Node.js (compliance engine)
PostgreSQL (users, reports, history)

## Compliance engine
Neo4j (88 rules knowledge graph)
Pinecone (regulation text vector search)
Claude API (plain English explanations)

## PDF generation
Puppeteer or PDFKit

## Deployment
Vercel (frontend + API)
Railway or Render (backend + databases)

## Repository structure
complyai/
├── frontend/     React / Next.js
├── backend/      Node.js compliance engine  
├── rules/        88 rules JSON files
├── docs/         Team documentation
└── design/       Figma links and assets
