# ComplyAI

EU AI Act compliance checker for hiring AI companies.

## What it does
Answer 5 questions about your hiring AI. 
Get an instant gap analysis with citations. 
Download an audit-ready PDF report.

## MVP Scope
- Framework: EU AI Act only
- Domain: Hiring AI — resume screening
- Target: Providers and deployers of resume screening AI

## Tech Stack
- Frontend: React / Next.js
- Backend: Node.js
- Graph DB: Neo4j
- Vector DB: Pinecone
- AI: Claude API
- Auth + Data: PostgreSQL
